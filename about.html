Про сервіс

Навчальний бізнес-проєкт «Твій кур'єр»

Сайт демонструє структуру з 5 HTML-сторінок, меню навігації та інтерактивний калькулятор. Усі розрахунки і динаміка виконуються на стороні клієнта через JavaScript.

Що реалізовано

  • Міста та тарифи у масивах JavaScript.
  • Калькулятор без перезавантаження сторінки.
  • Єдиний стиль оформлення CSS на всіх сторінках.
  • Авторство вказано у футері та у верхньому коментарі кожного HTML-файлу.

Правила

Без PHP, баз даних, файлів або API. Сайт відкривається локально та підходить для публікації на GitHub Pages.
